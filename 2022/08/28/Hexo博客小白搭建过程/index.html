Hexo博客小白搭建过程

环境配置

  • node安装
  • npm安装
  • cnpm管理器（taobao镜像源）

博客搭建

  1. 安装hexo框架
  2. 创建博客目录
  3. 生成并初始化博客
  4. 启动博客本地化服务（本地访问地址：http://localhost:4000）
  5. 创建一篇文章测试博客

配置ssh密钥

  1. 任意文件夹打开git bash
  2. 利用命令”ssh -keygen -t rsa -C ‘邮箱地址’ “绑定ssh（为更新博客方便不设密码）
  3. 在c盘的user/管理员的文件夹中找到”id_rsa.pub”文件，利用记事本打开，该内容为密钥

该密钥设置后不可见，所以一定要拍照或以其他方式保存一下

  1. 在github的setting中设置密钥，将刚才的密钥直接复制粘贴过来
  2. 密钥安装成功后，利用”ssh -T git@github.com“命令验证ssh密钥是否安装成功

发布博客

  1. github创建一个新的仓库
  2. blog目录下安装git部署插件
  3. 配置config文件（使用ssh地址，branch老版为master，新版为main）
  4. 将博客本地文件上传到github仓库中

github连接可能总有问题，uu某速器中搜索学术资源可进行加速，可能会改善连接问题
