大作文框架句型（四类）

优缺点（disadvantage & advantage）

P1

  • 引出趋势：

    • There is a growing trend in the modern society that…
    • In the modern society, an increasing number of people…
    • In the modern society, it has become increasingly common that…

  • 表明立场

    • From my perspective, this is mainly a positive/negative development.

P2 (让步 + 反驳)

  • 让步观点

    • Admittedly, …
    • There is no denying that, …
    • Of course, …

  • 让步观点补充

    • In addition, …
    • Besides, …
    • Also, …

  • 反驳

    • However, …

P3 (立论)

  • 立论观点

    • On the upsides/downsides, …

  • 立论观点补充

    • In addition, …
    • Besides, …
    • Also, …

P4

  • conclusion
    • In conclusion, although … , …

同意与否 (agree & disagree)

P1

  • 转述观点

    • It is believed that…

  • 表明立场

    • Personally, I tend to/firmly disagree with this view…

P2 (让步)

  • 让步观点

    • Admittedly, …
    • There is no denying that, …
    • Of course, …

  • 让步观点补充

    • In addition, …
    • Besides, …
    • Also, …

P3 (反驳)

  • 反驳观点

    • However, …
    • Apart from that, …

  • 反驳观点补充

    • In addition, …
    • Besides, …
    • Also, …

P4

  • conclusion
    • In conclusion, although … , …

讨论 (discuss)(结构一：万能结构)

P1 (引出争议点)(不给出观点)

  • People are divided on (whether…)

P2 (讨论甲方)

  • 段落开头

    • Some people think that…
    • This is beacause…

  • 观点补充

    • In addition, …
    • Besides, …
    • Also, …

P3 (讨论乙方)

  • 段落过渡 + 开头

    • Others, on the other hand, argue that…

  • 观点补充

    • In addition, …
    • Besides, …
    • Also, …

P4 (观点)

  • From my perspective, …

讨论 (discuss)(结构二：同意与否)

P1

  • 改写不同意的观点

  • Personally, I disagree with this view.

P2 (结构 = 同意与否)

P3 (结构 = 同意与否)

P4 (结构 = 同意与否)

说明文

P1

  • 转述社会问题

    • In the modern society, there is a worrying trend that…
    • In the modern society, it has become increasingly common that…
    • In the modern society, it has become a social issue that…

  • 交代写作意图

    • In this essay, I intend to discuss the underlying causes of this problem and come up with some possible solutions.

P2 (论述reason)

  • 段落开头

    • One of the most obvious reasons is +n./that从句…

  • 段落补充

    • Another key factor/root cause is…

P3 (论述solution)

  • 段落开头

    • The issue is so complex that there is no easy solution to it. However, in order to address this problem, I think a combined effort from the community and individuals can make some differences.

  • 补充一

    • First, …

  • 补充二

    • More importantly, …

P4 (conclusion)(时间不够可以不写)

  • In conclusion, 升华

核心的论证思路

具体化

  • 审题时做“概念的界定”

  • 主体段在解释观点是具体化“抽象概念”

What/Why

  • what：名词的概念和特点

  • why：梳理逻辑链(reasoning)

核心的论证方法（四种）

  • reason

  • example

  • if

  • make comparison

    • compared with…
    • unlike…
    • In the past, …
    • … , But now…

因果句式

5个明连接

1. 连词

  • because = as = since
  • so

2. 介词短语

  • because of = for the sake of = owing to = thanks to
  • due to

3. 副词

  • Therefore, = Consequently,
  • As a result,

4. 名词

  • consequence = result = outcome
  • main/major/obvious/primary reason
  • root/underlying cause
  • key/decisive factor

5. 动词

  • cause
  • lead to
  • contribute to
  • be responsible
  • it is … that should be blamed

5个暗连接

1. 状语从句

  • 条件状语从句：if…/unless/without…
  • 目的状语从句：in order to…
  • 方式状语从句：by/through/via…

2. A与B相关

  • A is related to B
  • be (closely) related to = be linked to = be associated with
